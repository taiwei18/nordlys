# 关于

TaiWei 是一个专注前端与工程实践的中文技术站点，内容涵盖 Astro 建站、性能优化、工程化协作与个人项目总结。本站强调可读性与可维护性，尽量保持结构清晰、内容可复用。

## 站点结构

站点包含 [首页](/)、[博客](/posts/)、[项目](/projects/)、[关于](/about/) 与 [404](/404/) 页面。新增内容只需在 `src/pages`、`src/content/posts` 或 `src/content/projects` 中创建新的 `.md` 文件即可。

## 配置与扩展

站点基础配置集中在 `src/theme.config.ts`。更多用法与扩展示例可在 [博客](/posts/) 中查看，包括主题配置、组件写法、内容组织方式与 SEO 细节设置。
